UF CLAS UFL Brand
=================

Description
-----------

WordPress child theme, based on the 2016 University of Florida Brand design. Maintained by CLAS IT.

- Author: Al Kirby
- Plugin URI: http://clas.ufl.edu/comms

Please contact [CLAS IT](https://it.clas.ufl.edu/) for more information.

Requirements
------------
- WordPress 4+
